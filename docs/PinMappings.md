## Additional Pin Notes

**Not exposed on headers:**
- D8 = LoRa NSS (Chip Select CS)
- D4 = LoRa Reset  
- D3 = LoRa DIO

**WiFi pins:**
- D8 = WiFi
- D7 = WiFi
- D4 = WiFi
- D2 = WiFi
